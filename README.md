Assessment & Doubt Tagging System


📌 Project Overview


This project is a web-based application developed using Flask that evaluates student answers using Natural Language Processing (NLP) techniques. It calculates similarity between student answers and correct answers and identifies areas where students may have doubts.


Features


1. Student login and dashboard


2. Teacher dashboard for monitoring performance


3. Automatic answer evaluation using NLP


4. Cosine similarity for answer comparison


5. Doubt identification system


6. Clean UI using HTML & CSS.


Technologies Used:


Python (Flask)


HTML, CSS


Machine Learning (TF-IDF, Cosine Similarity)
